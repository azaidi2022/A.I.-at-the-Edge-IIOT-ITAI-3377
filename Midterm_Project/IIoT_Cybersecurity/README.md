# Midterm Project – IIoT Cybersecurity Smart Factory

## Overview

The goal of this project was to design a cybersecurity plan for an AI-integrated Industrial Internet of Things (IIoT) smart factory.

As IIoT systems become more advanced, they also become more vulnerable to cyber threats. This project focused on identifying those risks and designing a system that is both functional and secure.

---

## Problem

IIoT systems connect multiple components such as:

- Sensors  
- Cameras  
- Edge devices  
- Cloud systems  
- AI models  

This connectivity introduces security risks, including:

- Unauthorized access  
- Data interception  
- System manipulation  
- AI model attacks  

The challenge was to design a system that protects against these threats.

---

## Approach

The system design included:

### Architecture
- Sensors collect machine data  
- Data flows to an edge gateway  
- AI models analyze data locally  
- Results are sent to dashboards and cloud storage  

### Vulnerability Analysis
Identified risks in:

- Devices (default passwords, outdated firmware)  
- Network (lack of encryption, poor segmentation)  
- Data (lack of protection and validation)  
- Applications (weak authentication, insecure APIs)  
- AI models (data poisoning, adversarial inputs)  
- Human factors (phishing, weak passwords)  

---

## Defense Strategy

A layered security approach was designed, including:

- Multi-factor authentication (MFA)  
- Role-based access control (RBAC)  
- Encryption (TLS, VPN)  
- Network segmentation and firewalls  
- Intrusion detection systems  
- Secure API and software practices  
- Continuous monitoring and logging  

---

## Lessons to be Learnt

This project shows one that security is not just one feature—it must be built into every layer of a system.

Some key takeaways:

- IIoT systems have many entry points for attacks  
- Human factors are often one of the biggest risks  
- AI systems introduce new types of vulnerabilities  
- Security must be proactive, not reactive  

---

## Challenges

- Identifying vulnerabilities across multiple system layers  
- Designing a complete security strategy  
- Understanding how AI systems can be attacked  

---

## Real-World Relevance

Cybersecurity is critical in industries such as:

- Manufacturing  
- Energy systems  
- Healthcare  
- Smart infrastructure  

A failure in these systems can lead to financial loss, safety risks, and operational failure.

---

## Conclusion

This project demonstrated that securing an IIoT system requires a comprehensive and layered approach.

By combining authentication, encryption, monitoring, and AI-specific protections, it is possible to significantly reduce system vulnerabilities.
